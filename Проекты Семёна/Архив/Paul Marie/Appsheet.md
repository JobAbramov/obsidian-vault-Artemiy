### Проект приложения
Приложение - небольшое CRM для выездных сотрудников, где они будут работать с задачами и отчётами.

- Первый раздел - это список с сегодняшними задачами, где менеджер может открыть детали по задаче, заполнить данные и закрыть. Задачи будут нескольких типов: по заселению/выселению и отчёты по затратам;

- Задача будет в виде формы, где единственным полем, доступным для редактирования - это поле, которое валидно для конкретного типа задачи. Там же можно поменять статус задачи;

- Ещё будет календарь, где будут показаны все задачи на пользователя.

Результат: приложение сделать возможно. ~10 часов
